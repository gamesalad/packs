packs
=====
A open source BoxJS/CSS/Etc clone.

Usage
-----

    $ cp config.css.ru config.ru
    $ rackup



or

    $ cp config.js.ru config.ru
    $ rackup

or

    $ cp config.img.ru config.ru
    $ rackup
